8.9 Tutorials: other examples

» Other extensions examples

You will find a few more Actionscript server side examples in the sfsExtensions/ folder.

simpleSetInterval.as   Shows how to use the setInterval() command on the server side
variablesTest.as   Shows how to access Room Variables from a server extension

We don't provide specific tutorials for these examples as they are very simple and the source code is well commented.

In order to check the examples, just plug them in the "simpleChat" Zone by adding a new <extension></extension> tag in the config file.
Then simply reload the Zones from the AdminTool.
