myUserName

Availability:

Flash Player 6.0
SmartFoxServer Lite / Basic / Pro

Usage:

smartFox.myUserName

Description:

A string representing the name we're currently using in the server.
The variable is available only after a successful login.

Parameters:

none      

Returns:

Nothing.

Example:

trace("I logged in as: " + smartFox.myUserName)

See also:

login()
