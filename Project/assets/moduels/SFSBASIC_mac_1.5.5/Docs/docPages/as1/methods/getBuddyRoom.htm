getBuddyRoom()

Availability:

Flash Player 6.0
SmartFoxServer Basic / Pro

Usage:

smartFox.getBuddyRoom(buddy)

Description:

Ask the server in which room(s) is currently located the buddy passed in the parameter.

Parameters:

buddy   the buddy object

NOTE: A buddy is obtained by extracting them from the smartFox.buddyList array

Fires the onBuddyRoom event

Returns:

Nothing.

Example:

smartFox.getBuddyRoom(buddy)
