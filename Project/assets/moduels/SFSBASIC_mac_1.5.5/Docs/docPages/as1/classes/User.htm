User

Availability:

Flash Player 6.0
SmartFoxServer Basic / Pro

Description:

The User class is used internally by the SmartFoxClient object to store the properties of each user.
Also User objects are returned by different methods and events of the API. Below follows a list of public methods available:

Public Methods:

getId()   returns the user Id
getName()   returns the user name
getVariable(varName)   returns a user variable
getVariables()   returns a list of all the user variables
isSpectator()   returns true if the user is a spectator in the game room
