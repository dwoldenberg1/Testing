onRandomKey()

Availability:

Flash Player 6.0
SmartFoxServer Pro

Usage:

smartFox.onRandomKey(theKey)

Description:

The event is fired in response to a getRandomKey() request. The server responds sending the unique string key.
For more details please check the Secure Login tutorial.

Parameters:

theKey   the unique random string key

Returns:

nothing

Example: 

smartfox.onRandomKey = function(theKey)
{
	trace("Here's the secret key: " + theKey)
}
