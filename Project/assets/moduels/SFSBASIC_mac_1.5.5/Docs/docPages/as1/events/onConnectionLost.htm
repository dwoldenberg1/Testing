onConnectionLost()

Availability:

Flash Player 6.0
SmartFoxServer Lite / Basic / Pro

Usage:

smartFox.onConnectionLost()

Description:

The event is fired when the connection with the server is closed. ( Either from the client or from the server )

Parameters:

none    

Returns:

nothing

Example: 

smartFox.onConnectionLost = function()
{
	trace("Connection was lost")
}

See also:

onConnection()
