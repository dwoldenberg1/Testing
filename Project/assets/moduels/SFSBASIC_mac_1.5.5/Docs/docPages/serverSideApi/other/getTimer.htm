getTimer()

Availability:

SmartFoxServer PRO 1.2.1

Usage:

getTimer()

Description:

Return the current time in milliseconds

Parameters:

none      

Returns:

Return the current time in milliseconds

Example:

// Simple timed benchmark test
var t1 = getTimer()

var list = []
for (i = 0; i < 10000; i++)
	list[i] = i

var t2 = getTimer()

trace("Test took " + (t2 - t1) + " ms.")
