trace()

Availability:

SmartFoxServer PRO 1.2.1

Usage:

trace(traceMessage)

Description:

Outputs the message to the SmartFoxServer console and to the Admininistrator Tool.
It can be useful for remote debugging when developing Extensions.

Parameters:

traceMessage     - A text message

Returns:

nothing.

Example:

trace("Hello World!")
