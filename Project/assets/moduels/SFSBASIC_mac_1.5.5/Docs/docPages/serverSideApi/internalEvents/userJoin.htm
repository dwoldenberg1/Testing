Internal Event :: userJoin

Availability:

SmartFoxServer PRO 1.2.1

Event name:

userJoin

Description:

This event is fired each time a user is joined in a room within the current Zone.
The event is available for both Zone Level and Room Level extensions.

Properties:

name   description type
user   The User object of the new client object
room   The Room object representing the room that the new client joined object

Example:

// Handle server events
function handleInternalEvent(evtObj)
{
	if (evtObj.name == "userJoin")
	{
		var r = evtObj.room
		var u = evtObj.user
		
		trace("User: " + u.getName() + " joined room: " + r.getName())
	}
}
