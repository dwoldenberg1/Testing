_server.getCurrentRoom()

Availability:

SmartFoxServer PRO 1.2.1

Usage:

_server.getCurrentRoom()

Description:

Returns the Room object where the extension is running. (If the extension is not at Room level you will get a null)

Properties:

none    

Returns:

Returns the Room object where the extension is running. (If the extension is not at Room level you will get a null)

Example:

/*
* Get the Room object
*/

var room = _server.getCurrentRoom()
trace("Extension is running in room: " + room.getName())
trace("Users in this room: " + room.getUserCount())
