_server.logger

Availability:

SmartFoxServer PRO 1.2.1

Usage:

_server.logger

Description:

The logger object sends logging informations and messages to SmartFoxServer. There are five different methods corresponding to the five logging levels available: FINER, FINE, INFO, WARNING, SEVERE.

Parameters:

_server.logger.finer(msg)   Logs a message with level FINER
_server.logger.fine(msg)   Logs a message with level FINE
_server.logger.info(msg)   Logs a message with level INFO
_server.logger.warning(msg)   Logs a message with level WARNING
_server.logger.severe(msg)   Logs a message with level SEVERE

msg is a String message

Returns:

nothing

Example:

_server.logger.warning("This is a WARNING loge message")
