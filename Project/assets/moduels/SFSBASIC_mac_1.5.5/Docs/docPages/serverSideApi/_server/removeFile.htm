_server.removeFile()

Availability:

SmartFoxServer PRO 1.2.1

Usage:

_server.removeFile(fileName)

Description:

Delete a file or an empty folder.

Properties:

fileName   the name of the file or directory to remove

Returns:

True if the file/directory was deleted successfully

Example:

Create a directory:

// Let's try to delete a file called temp.txt

var ok = _server.removeFile("temp.txt")

if (ok)
	trace("File removed")
else
	trace("Could not remove file!")
