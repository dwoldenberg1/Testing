_server.fileExist()

Availability:

SmartFoxServer PRO 1.2.1

Usage:

_server.fileExist(fileName)

Description:

Checks if a file or directory exist at the given path.

Properties:

fileName   the file or directory name to check

Returns:

True if the file / directory exist

Example:

/*
* Check if the log folder exist
*/

var exist = _server.fileExist("logs")

if (exist)
	trace("Found !")
else
	trace("Not found!")
