1.4 Upgrading

If you already have an older SmartFoxServer installed on your local machine or live server, it is important to back up some files before proceeding with an upgrade.

All the files that need to be backupped are located in the Server/ folder:

config.xml the main configuration file
licence.sfl backup this file if you own a commercial licence
bannedUsers.sfs backup this file if you want to save the current list of banned users
sfsExtensions/ backup this folder to save all your Actionscript extensions (PRO only)
javaExtensions/ backup this folder to save all your Java extensions (from version 1.4.0 and higher) (PRO only)
buddyLists/ backup this folder if you have persistent buddy lists enabled in your applications
logs/ backup this folder if you want to keep your server logs

Once these files are copied to a safe location you can either uninstall the current server manually (Linux/Unix) or automatically (Windows) and finally proceed with the installation of the new version.
