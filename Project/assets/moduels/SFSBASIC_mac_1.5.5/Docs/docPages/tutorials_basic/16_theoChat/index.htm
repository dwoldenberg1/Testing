5.16 Example: Theo Avatar Chat

» TheoWorld's Theo Avatar Chat

The TheoAvatarChat is an advanced chat application produced by TheoWorlds, featuring a dynamic isometric engine that allows unlimited customizations. You can find an example of the Theo Chat in the Examples/TheoAvatar/ folder.

TheoChat currently supports SmartFoxServer Basic and PRO. For more info about it, please visit the TheoWorlds website.
